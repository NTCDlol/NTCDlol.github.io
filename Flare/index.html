Flare — Язык

Flare — это динамический язык программирования, вдохновленный Python, Java, C и ASM. Он предназначен для создания мощных приложений на разных уровнях: от пользовательских скриптов до системных утилит и микроконтроллеров.

Гибкий синтаксис

Flare поддерживает два стиля записи кода:

  • Питоновский динамический режим:
    • if x > 10:
print("Больше десяти")
  • C-подобный статический режим:
    • if (x > 10) {
out("Больше десяти");

}

Управление памятью

Два режима:

  • FlameMemory (автоматическое): сборщик мусора, «слежение» за переменными и оптимизация использования оперативной памяти.
  • ManualMemory (статическое): вручную выделяйте и освобождайте память с помощью команд mem, frmem.

Работа с бинарными данными

Прямо в языке:

bin.16.value = 0xFFAA

bin.arm64.x = 0x12345678

Функции и модули

Определяйте функции разных битностей и подключайте библиотеки:

prog sum(a, b) f32:
return a + b

add Flare.MATH print(sqrt(9))

Условия, циклы и ошибки

  • if и for в двух стилях;
  • err("текст") для ошибок, warn("текст") для предупреждений;
  • allstop — аварийная остановка.

Подключение библиотек

Используйте репозиторий Fire:

add Flare.ALL

add Flare.NET from Flare.MATH add sqrt
